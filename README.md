# ml-studies

Personal machine learning study repository. Structured curriculum targeting a research internship.

---

## Tracks

Four parallel tracks per week:

| Track | Source | Focus |
|---|---|---|
| Theory | Jung – *ML: The Basics* | Concepts, ERM, generalization |
| Prob/Stats | Murphy – *Probabilistic ML* (selective) | Bayes, MLE, MAP, KL divergence |
| Practice | Géron – *Hands-On ML* + Karpathy *Zero to Hero* | sklearn, PyTorch, from-scratch implementations |
| Deliverable | Weekly notebook | End-to-end projects tracked with W&B |

---

## Curriculum

**Phase 1 — Foundations + Probability**  
ML landscape, hypothesis space, loss functions, empirical risk minimization, gradient descent (SGD, Adam), model validation and cross-validation. Probability core: Bayes rule, common distributions, MLE, MAP, bias-variance tradeoff.
